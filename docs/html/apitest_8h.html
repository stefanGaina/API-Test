This defines the main function and parses the arguments for an input file. Initializes API-Test and parses commands until "quit" or "q" is inputed by the user.

Parameters
titleThe text printed at the beginning of the line when waiting for the commands to be inputed in terminal and not file.
buffer_sizeThe maximum length of the inputed command (if exceeded it will be truncated).
handler_functionThis function is called everytime a new command has been successfully got.
print_help_functionThis function is called when "help" or "h" is inputted by the user.
Returns
void
